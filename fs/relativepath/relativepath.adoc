= relativepath

[source,lua]
----
function relativepath( from, to ) --> path
----

== Description

Returns a relative path between the paths _from_ and _to_.

== Parameters

from::
The source path.

to::
The destination path.

== Return Values

path::
The relative path between _from_ and _to_.
