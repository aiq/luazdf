= digitsum

[source,lua]

function digitsum( num ) --> sum
----

== Description

Returns the _sum_ of all digits of the given integer _num_.

== Parameters

num::
The integer where we want to build the _sum_.

== Return Values

sum::
The _sum_ of all digits of the given integer _num_.
